It can get literally soaked and bounce right back. It’s incredibly easy to care for; it’s receptive to neglect and papering. Change directory to last PWD

TIL this trick from a coworker: cd - expands to cd $OLDPWD. It’s effectively a “back” button when back is more complex than cd ../. The expansion of just - is specific to the cd builtin, but more generally, ~- expands to $OLDPWD as explained here. So you can use run commands like: echo ~- to print the last working directory cp ~-/* . to copy everything from the previous working directory to the current one Contact me

Change directory to last PWD

TIL this trick from a coworker: cd - expands to cd $OLDPWD. It’s effectively a “back” button when back is more complex than cd ../. The expansion of just - is specific to the cd builtin, but more generally, ~- expands to $OLDPWD as explained here. So you can use run commands like: echo ~- to print the last working directory cp ~-/* . to copy everything from the previous working directory to the current one Contact me
