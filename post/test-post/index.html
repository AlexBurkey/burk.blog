My First Blog Post

Hey, Alex here! It’s been a goal of mine to have a blog/website for little posts and thoughts that I have about things. So here it is!

I’m using the static-site generator Hugo to build this. I just have to configure the theme to make the site look nice and I can write posts using markdown for formatting. I’m hosting it using GitHub Pages which is super easy and I highly recommend for static sites.

I picked Hugo over Jekyll for no particular reason at all. It’s a wildly popular choice alongside Hugo and seems just as capable. The only major functional difference I’ve experienced is that Hugo offers LiveReload which means that I can make updates to content and configurations while the Hugo server is running and I can see those updates just by refreshing the browser. No need to rebuild the site after each update.

Check out my About page or at some point in the future my posts page for more stuff!
